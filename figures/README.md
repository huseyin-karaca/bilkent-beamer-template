# Figures Directory

Place your presentation figures (images, diagrams, plots) in this directory.

Supported formats: PDF, PNG, JPG

Example usage in `main.tex`:
```latex
\begin{figure}
    \centering
    \includegraphics[width=0.8\textwidth]{your_figure.pdf}
    \caption{Your figure caption}
    \label{fig:your_label}
\end{figure}
```
